# Node Calculator
Simple calculator so I can learn the basics of command line utilities with NodeJS

## Commands
The structure of a command is: `node app <operator> <operrand1> <opporand2>`  
Example: `node app add 2 2` will output `4`

### Add
`add` || `-a`

### Subtract
 `sub` || `-s`

### Multiply
`mult` || `-m`

### Divide
`div` || `-d`
